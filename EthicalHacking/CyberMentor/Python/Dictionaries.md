```python
dict = {"one":1, "two":2, "three":3} # dict is a key-value pairs storage
print(dict["one"])
print("one" in dict) #checks if there is a key "one" in a dictionary

dict["four"] = 4 # adds new key-value pair in dictionary

for key in dict.keys(): #iterates through dictionary keys
    print(key)

for value in dict.values(): #iterates through dictionary values
    print(value)
```
